# Hugo Usage

## Create A New Post

```bash
hugo new posts/{name-of-the-post}.md
```

## Start A Hugo Server

```bash
hugo server -D
```

- ``-D``: With draft enabled.

## Build The Static Page

```bash
hugo -D
```
